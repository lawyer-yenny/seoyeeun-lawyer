### 1️⃣ Google 검색 등록

1. [Google Search Console](https://search.google.com/search-console) 접속
2. **속성 추가** 클릭
3. **URL 접두어** 선택
4. 사이트 주소 입력: `https://lawyer-yenny.github.io/seoyeeun-lawyer/`
5. **소유권 확인** (HTML 파일 업로드 또는 메타태그 방식)
6. **Sitemaps** 메뉴에서 사이트맵 제출:
   ```
   https://lawyer-yenny.github.io/seoyeeun-lawyer/sitemap.xml
   ```

**소요 시간:**
- 등록: 즉시
- 검색 노출: **1주~1개월**
